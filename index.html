Python Base Skill (I)

  這篇要簡單告訴大家有關Python的基本語法，只要知道這些基本的語法，大致上就能完成所有的事情了。
  首先，對於沒學過程式的同學們，我們先簡單介紹一下有關程式的基本知識。

Type

  在程式中，每一個東西都有自己的 Type(中文可以翻作"型別")，
比如數字 1 就屬於 Integer(整數，在大部分程式語言裡會以 int 簡寫)。
  一般說，不同Type的東西不會直接有互動，就像我們不會問一秒加上八顆蘋果是多少。
  由這裡可以看出，Type 在程式語言中扮演了類似 單位 的角色。
  下面我們給大家參考在python裡較常出現的Type:
python常用Type
Type 全文 中文 範例 備註
int Integer 整數 50 整數
float Float 浮點數 5.34 帶有小數點的數值，有效位數為16位
str String 字串 "string" 字串就是一串字的意思，須注意打程式時要以 " 符號將字串包起來
bool Boolean 布林 True / False 只用來記錄是或否的Type，注意字首需大寫

輸出

  對第一次接觸程式的人來說，最關心的就是它到底有沒有在動，也就是可以跟程式有所互動。
  在python裡，輸出的第一步可以請大家在程式裡打上code:
  
  print("balabala")
  
  然後執行，你就會在執行視窗中看到output:
  
  balabala

  由這個範例，大家應該可以看的出來，只要打上 print() 並把想要打印出來的字串放在括弧中，
就可以顯示字了!! (注意，記得字串需要以 "想要印出來的字" 的形式打出來喔!)

簡單數學計算

  再來我們介紹簡單的數學計算。
  在python裡，加法和減法所使用的算符就是一般的 + & -；
而乘除的部分則使用 * & / 兩個符號作為算符。
另外python還可以用 ** 來做指數運算。
  以下整理一個表格:
常用運算符號
算符 意義 範例 結果 備註
+ 加法 3+2 5
- 減法 3-2 1
* 乘法 3*2 6
/ 除法 3/2 1.5 在python裡，做除法時會自動把 int 轉換成 float
** 指數運算 3**2 9 再算符前面的是底數，算符後面的是指數，指數可接受 float type
