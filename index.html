Python Base Skill (I)

  這篇要簡單告訴大家有關Python的基本語法，只要知道這些基本的語法，大致上就能完成所有的事情了。
  首先，對於沒學過程式的同學們，我們先簡單介紹一下有關程式的基本知識。

輸出

  對第一次接觸程式的人來說，最關心的就是它到底有沒有在動，也就是可以跟程式有所互動。
  在python裡，輸出的第一步可以請大家在程式裡打上code:
  
  print("balabala")
  
  然後執行，你就會在執行視窗中看到output:
  
  balabala

  由這個範例，大家應該可以看的出來，只要打上 print() 並把想要印出來的字以 "想要印出來的字" 的形式打在括弧中，
就可以顯示字了。

Type

  在程式中，每一個東西都有自己的 Type(中文可以翻作"型別")，
比如數字 1 就屬於 Integer(整數，在大部分程式語言裡會以 int 簡寫)。
  一般說，不同Type的東西不會直接有互動，就像我們不會問一秒加上八顆蘋果是多少。
  由這裡可以看出，Type 在程式語言中扮演了類似 單位 的角色。
