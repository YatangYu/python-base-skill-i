Type

	在程式中，每一個東西都有自己的 Type(中文可以翻作"型別")，比如數字 1 就屬於 Integer(整數，在大部分程式語言裡會以 int 簡寫)。
一般來說，不同Type的東西不會直接有互動，就像我們不會問一秒加上八顆蘋果是多少。由這裡可以看出，Type 在程式語言中扮演了類似 單位 的角色。 

	再來，要教大家python定義變數的方式。假如我們要做的計算中有一些數值是有它的意義的，像是經過2秒的時間，這裡的2就代表著經過多少的時間。
物理上時間通常會以t作為代號，那麼我們就可以在程式裡打上code:

t=2
  
這樣就能在程式中創造一個叫做 t 的變數，而它的值是 2 。
以上是關於Type的簡單介紹，下面我們給大家參考在python裡較常出現的Type:
python常用Type
Type 全文 中文 範例 備註
int Integer 整數 50 整數
float Float 浮點數 5.34 帶有小數點的數值，有效位數為16位
str String 字串 "string" 字串就是一串字的意思，須注意打程式時要以 " 符號將字串包起來
bool Boolean 布林 True / False 只用來記錄是或否的Type，注意字首需大寫

輸出

	對第一次接觸程式的人來說，最關心的就是它到底有沒有在動，也就是可以跟程式有所互動。
在python裡，輸出的第一步可以請大家在程式裡打上code:
  
print("balabala")
  
然後執行，你就會在執行視窗中看到output:
  
balabala

由這個範例，大家應該可以看的出來，只要打上 print() 並把想要打印出來的字串放在括弧中，就可以顯示字了!!
   (注意，字串需要以 "想要印出來的字" 的形式打出來，print也可以印其他Type的變數)

輸入

	與輸出相對的，輸入也是與程式互動很重要的一環。
假設想要與電腦做個小互動，我們把名字輸入，然後讓電腦印出"Hello (我的名字)"可以寫code:
  
name=input("請輸入名字:")
print("Hello",name)
  
執行後會在執行視窗看到:

請輸入名字:

接著，你就可以在執行視窗裡輸入自己名字(如果用dog)，輸入完成按下Enter，就會跳出:

Hello dog

	範例可以看到，input()函式的括胡中可以放入想要在執行視窗中顯示的提示文字(如"請輸入一個數字")；
再來，我們在執行視窗輸入的東西會被讀到我們要的變數裡(範例中是name)

簡單數學計算

	再來我們介紹一些簡單的數學計算。在python裡，加法和減法所使用的算符就是一般的 + & -；
而乘、除的部分則使用 * & / 兩個符號作為算符；另外%可以計算餘數，python還可以用 ** 來做指數運算。
以下整理一個表格:
常用運算符號
算符 意義 範例 結果 備註
+ 加法 3+2 5
- 減法 3-2 1
* 乘法 3*2 6
/ 除法 3/2 1.5 在python裡，做除法時會自動把 int 轉換成 float
% 餘數運算 3%2 1 算符前面的是被除數，算符後面的是除數，除數、餘數可以是非整數，但運算結果可能與想像不同(電腦邏輯問題)；建議都做整數運算
** 指數運算 3**2 9 算符前面的是底數，算符後面的是指數，指數可接受 float type
